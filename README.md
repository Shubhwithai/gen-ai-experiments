## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:
- **Python 3.10+** (Python 3.11 recommended)
- **pip** or **uv** package manager
- **Git** for version control
- **Jupyter Notebook** or **JupyterLab** (for notebook experiments)

**API Keys** (depending on the project):
- OpenAI API key
- Anthropic API key
- Google AI API key
- Other provider-specific keys

### Quick Start

1. **Clone the repository**

    ```bash
    git clone https://github.com/buildfastwithai/gen-ai-experiments.git
    cd gen-ai-experiments
    ```

2. **Choose your learning path:**

    **For Jupyter Notebooks (Learning):**
    ```bash
    # Install Jupyter
    pip install jupyter

    # Navigate to 100-os-libraries or any notebook directory
    cd 100-os-libraries

    # Launch Jupyter
    jupyter notebook
    ```

    **For Streamlit Apps (Production Apps):**
    ```bash
    # Navigate to your desired app
    cd ai-apps-collection/your-desired-project

    # Install dependencies
    pip install -r requirements.txt

    # Run the app
    streamlit run app.py
    ```

3. **Set up environment variables** (create a `.env` file):

    ```env
    OPENAI_API_KEY=your_openai_api_key
    ANTHROPIC_API_KEY=your_anthropic_api_key
    GOOGLE_API_KEY=your_google_api_key
    # Add other keys as needed
    ```

4. **Follow project-specific instructions** in each project's `README.md` file for detailed setup and configuration.
